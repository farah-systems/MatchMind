# MatchMind
MatchMind - a machine learning system to predict EPL match outcomes and scores using historical match statistics, feature engineering, and model evaluation.

## Project Overview
this project predicts english premier league match outcomes and scores using machine learning models trained on historical match statistics.

## Objectives
- Predict match outcomes (win/draw/loss)
- Predict home & away goals
- Analyse which factors influence results
